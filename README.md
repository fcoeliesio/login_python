# 🔐 Flask MVC Login System

Sistema completo de autenticação desenvolvido com **Flask**, utilizando arquitetura **MVC (Model-View-Controller)**.

O projeto implementa cadastro de usuários, login com sessão, proteção de rotas e estrutura organizada para escalabilidade.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Login com sessão
- Logout seguro
- Perfil de usuário protegido
- Senhas criptografadas
- Validação de autenticação
- Arquitetura MVC
